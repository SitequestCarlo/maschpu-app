Die **S**chacht-**E**inlauf**t**onne (SET) ist eine mobile Vorrichtung, die eine Vielzahl von Einsatzmöglichkeiten bietet:

- Einleiten von Wasser in Schächte
- Freies Ausleiten als "Quelltopf"
- Flachansaugen (mittels eines Einlaufgitters)
- Sedimentfreies Ansaugen aus Gewässern
- Fördern von Schöpfgut
- Kühlen von Acetylenflaschen

## Einleiten in Schächte

Beim direkten Einleiten in Schächte knicken Schläuche leicht ab und reduzieren die Förderleistung,
während der ungebremste Strahl die Schachtwand ausschwemmt und das umgebende Erdreich destabilisiert.

<figure>
  <Ausspuelung alt="Zerstörte Straße durch Unterspülen beim Einleiten in einen Schacht" />
  <figcaption>
    Zerstörte Straße durch Unterspülen beim Einleiten in einen Schacht
  </figcaption>
</figure>

Die Schachteinlauftonne führt mehrere Leitungen horizontal in den Schacht, bremst den Strahl und leitet ihn
drucklos nach unten ab. Gleichzeitig verschließt die SET den Schacht, sodass weder Personen noch Material hineinfallen kann.

## Freies Ausleiten als "Quelltopf"

Umgedreht eingesetzt dient die SET als freier Auslauf: Sie nimmt mehrere Schlauchleitungen auf, bricht den Strahl und schützt Ufer oder Untergründe vor Ausspülungen.
Eine Auslaufplane von mindestens 4x4 Metern verhindert zusätzlich das Durchweichen des Bodens, während schwere Partikel in der SET zurückbleiben.

## Flachansaugen

Mittels eines Einlaufgitters kann die SET auch zum Flachansaugen verwendet werden. Das Gitter verhindert das Ansaugen von groben Partikeln und hebt die SET gleichzeitig an.
