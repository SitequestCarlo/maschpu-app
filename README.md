# MaschPu – WebApp für Maschinisten von Pumpen

MaschPu ist eine Progressive Web App (PWA) rund um das Thema Pumpen und Pumpentechnik im THW / Feuerwehr-Umfeld. Die App funktioniert vollständig offline und bietet Theorie, Rechner und Pumpen-Datenblätter für den Einsatz und die Ausbildung.

## Features

- Vollständige Offline-Funktionalität nach dem ersten Besuch
- Hydraulische Rechner (Fördermenge, Druckleitung, Saughöhe, etc.)
- Pumpen-Datenblätter (Elektro-, Motor- und Großpumpen)
- Theorie zu Strömungslehre, Einsatztaktik und Pumpentechnik
- Automatische Updates im Hintergrund

## Lokale Entwicklung

```shell
npm install
npm start
```

## Build

```shell
npm run build
npm run serve
```
